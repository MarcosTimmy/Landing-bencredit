# Bencredit Landing Page

Proyecto estático organizado para abrir directamente en Visual Studio Code.

## Estructura

- `index.html`: contenido principal.
- `css/styles.css`: diseño y adaptación móvil.
- `js/app.js`: menú móvil, formulario y año automático.
- `assets/`: carpeta preparada para logo, imágenes e iconos.

## Cómo abrir

1. Abre la carpeta `bencredit-landing` en Visual Studio Code.
2. Abre `index.html` en el navegador o usa la extensión **Live Server**.

## Antes de publicar

Reemplaza en `index.html`:

- `https://app.bencredit.com` por la URL real de la app.
- `info@bencredit.com` por el correo oficial.
- El número telefónico de ejemplo.
- La letra B por el logo oficial cuando esté disponible.

El formulario es visual y todavía no envía datos. Debe conectarse a una API, servicio de formularios o correo.
